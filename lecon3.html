Leçon 3 : Le Modèle TCP/IP

Découvrez le protocole réel qui fait fonctionner Internet aujourd'hui.

Qu'est-ce que le TCP/IP ?

Alors que le modèle OSI est un modèle de référence théorique, le modèle TCP/IP (Transmission Control Protocol/Internet Protocol) est l'architecture réelle utilisée pour toutes les communications sur le web.

Les 4 Couches du modèle TCP/IP

Le TCP/IP simplifie le processus en regroupant certaines fonctions en 4 couches clés :

N° Couche Protocoles associés
4 Application HTTP, HTTPS, FTP, DNS
3 Transport TCP (Fiable) et UDP (Rapide)
2 Internet IP (Adressage), ICMP (Ping)
1 Accès Réseau Ethernet, Wi-Fi, MAC

Point Clé : TCP vs UDP

  • TCP : Garantit que les données arrivent sans erreur (utilisé pour le Web, Email).
  • UDP : Envoie les données sans vérifier la réception (utilisé pour le Streaming, Jeux vidéo).
