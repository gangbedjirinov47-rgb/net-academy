Leçon 7 : Les Supports de Transmission

Découvrez les différents médias utilisés pour transporter les signaux binaires d'un point à un autre.

1. Les Supports Cuivrés (Signaux électriques)

Le câble Paire Torsadée (RJ45) est le plus commun. Il est classé par catégories (Cat 5e, 6, 7) selon sa capacité à transporter des données sans interférences.

2. La Fibre Optique (Signaux lumineux)

Composée d'un fil de verre ou de plastique très fin, elle transmet les données sous forme d'impulsions lumineuses. Elle offre une vitesse immense sur de très longues distances.

3. Les Supports Sans Fil (Ondes radio)

Le Wi-Fi et le Bluetooth utilisent des ondes électromagnétiques. C'est pratique pour la mobilité, mais le signal peut être affaibli par les obstacles physiques (murs).
