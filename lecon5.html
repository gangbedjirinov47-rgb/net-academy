Leçon 5 : Les Équipements du Réseau

Découvrez les appareils physiques qui permettent de connecter et de diriger les données.

1. Le Switch (Commutateur)

Le Switch est l'équipement central d'un réseau local (LAN). Il relie plusieurs ordinateurs entre eux. Contrairement au Hub, il est intelligent : il envoie les données uniquement au destinataire grâce à son adresse MAC.

2. Le Routeur

Le Routeur est le "chef de gare" du réseau. Son rôle est de relier deux réseaux différents (par exemple votre maison et Internet). Il choisit le meilleur chemin pour les données en utilisant les adresses IP.

3. Le Point d'Accès Sans Fil (Access Point)

Il permet aux appareils mobiles de se connecter au réseau filaire via des ondes radio (Wi-Fi).

Comparaison : Switch vs Routeur

Caractéristique Switch Routeur
Niveau OSI Couche 2 (Liaison) Couche 3 (Réseau)
Identification Adresse MAC Adresse IP
Usage Réseau local (Interne) Interconnexion (Externe)
